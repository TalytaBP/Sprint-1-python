Projeto acessibilidade para daltônicos

Solução proposta

Criar uma interface mais acessível para daltônicos e a criação de um chatbot para auxiliar o usuário a navegar pelas soluções entregues pela Salesforce

Paginas

O projeto será uma landing page do portal da Salesforce que com opções de acessibilidade para daltônicos e um chatbot para auxiliar o usuário a encontra as soluções desejadas

acessibilidade

A landing terá um botão aonde o usuário poderá escolher o temas de cores que melhor se adaptam para o seu tipo de daltonismo

Chatbot

o chatbot ira auxiliar o usuário a encontrar o que esta procurando na pagina
